# 🌸 Website Dear Darlene (Versi Baru — Multi-Halaman)

Desain baru: **cream base + maroon aksen**, tombol rounded, editorial.
5 halaman terpisah: Home · Koleksi · About · News · Order.

## Isi folder
- `index.html` (Home) · `koleksi.html` · `about.html` · `news.html` · `order.html`
- `styles.css` (tampilan) · `shared.js` (footer & animasi)
- folder `images/` (semua foto)

> PENTING: semua file harus tetap dalam SATU folder yang sama (jangan dipisah),
> karena halaman saling terhubung dan memakai styles.css + shared.js bersama.

## ⭐ Deploy ke Vercel (cara GitHub — direkomendasikan)
1. github.com → New repository → kasih nama `dear-darlene-web` → Create.
2. "uploading an existing file" → seret SEMUA isi folder ini (semua .html, .css, .js, dan folder images).
3. Commit changes.
4. vercel.com → Add New → Project → pilih repo → Deploy. Selesai!

## 🌐 Domain deardarlene.co
Vercel → project → Settings → Domains → ketik `deardarlene.co` → ikuti instruksi DNS.

## ✏️ Edit cepat (buka order.html, cari bagian CONFIG)
- `waNumber` → nomor WhatsApp
- `bank` → rekening
- `ongkir: 0` → ganti angka kalau mau pasang ongkir (otomatis masuk total)
- `prices` → ubah harga
